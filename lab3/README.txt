In this lab, I implemented the insertion sort and merge sort algorithms with metrics using the myCompare() and myCopy() functions.
I used make to compile the .c files whenever i was done with implementation, and i tested the algorithms using a worst case file.

Theoretical analysis:

Insertion Sort:
- Best case: Comparisons O(n), Copies O(n), Swaps O(n)
- Average: Comparisons O(n^2), Copies O(n^2), Swaps O(n)
- Worst: Comparisons O(n^2), Copies O(n^2), Swaps O(n)

Merge Sort:
- Best Case: Comparisons O(n log n), Copies O(n log n), Swaps O(n)
- Average: Comparisons O(n log n), Copies O(n log n), Swaps O(n)
- Worst: Comparisons O(n log n), Copies O(n log n), Swaps O(n)
